**Goal:** Build a new `BTC 15m Hedge` bot with its own backend engine, persistence, API routes, UI tab, and tests so it can assemble one hedged `UP + DOWN` pair per 15-minute BTC market using limit buys and hold the paired portion to expiry.

**Architecture:** Implement a new isolated module under `backend/src/btc15m-hedge/` that mirrors the existing `btc15m/` layout but tracks two legs instead of one directional position. Reuse shared Polymarket service, market resolution, pricing, user WebSocket, and budget infrastructure; expose new `/api/btc15m-hedge/*` routes and a dedicated frontend tab that polls the new status payload.

**Tech Stack:** TypeScript, Node.js, existing Polymarket CLOB integration, existing budget manager, Vite/React frontend, isolated `tsx` test scripts, `tsc --noEmit`

## Data Model Decisions

- New state file path: `data/btc15m-hedge-state.json`
- New engine phases: `stopped | running | auto_stopped`
- New cycle phases: `waiting_market | building_pair | paired_holding | unwinding | cycle_done | market_idle`
- New leg model per side:
  - active buy order
  - cumulative filled shares
  - cumulative filled cost
  - average entry
- Paired metrics:
  - `pairedShares = min(up.filledShares, down.filledShares)`
  - `pairedUpCost = min(up.filledShares, pairedShares) * up.avgEntryPrice`
  - `pairedDownCost = min(down.filledShares, pairedShares) * down.avgEntryPrice`
  - `combinedAverage = pairedShares > 0 ? (pairedUpCost + pairedDownCost) / pairedShares : null`
- Completed cycle row stores one row per market, not one row per fill

## Runtime Rules to Preserve

- Only one hedge cycle per market
- No re-entry after cycle completes or pair is assembled/unwound
- Partial fills are real; `matched` is not final fill
- At `< forceUnwindThresholdMin`, cancel remaining buys and sell only unpaired remainder
- Paired symmetric portion is held to market settlement
- Budget must count both legs at once

```ts
function derivePairState(upLeg: Btc15mHedgeLegState, downLeg: Btc15mHedgeLegState) {
  const pairedShares = roundShares(Math.min(upLeg.filledShares, downLeg.filledShares));
  if (pairedShares <= 0) {
    return {
      pairedShares: 0,
      pairedAvgUp: null,
      pairedAvgDown: null,
      combinedAverage: null,
      unpairedUpShares: roundShares(upLeg.filledShares),
      unpairedDownShares: roundShares(downLeg.filledShares),
    };
  }

  const pairedUpCost = roundUsd(pairedShares * (upLeg.avgEntryPrice ?? 0));
  const pairedDownCost = roundUsd(pairedShares * (downLeg.avgEntryPrice ?? 0));
  const combinedAverage = roundPrice((pairedUpCost + pairedDownCost) / pairedShares);
  return {
    pairedShares,
    pairedAvgUp: upLeg.avgEntryPrice,
    pairedAvgDown: downLeg.avgEntryPrice,
    combinedAverage,
    unpairedUpShares: roundShares(upLeg.filledShares - pairedShares),
    unpairedDownShares: roundShares(downLeg.filledShares - pairedShares),
  };
}
```

```ts
    leg.filledShares = roundShares(leg.filledShares + shares);
```

```ts
private async ensureDualBuyOrders(now: number): Promise<void> {
  if (this.state.cycle.completionLocked) return;
  await this.ensureBuyForLeg("up", this.state.market!.upTokenId, now);
  await this.ensureBuyForLeg("down", this.state.market!.downTokenId, now);
}

private async ensureBuyForLeg(side: "up" | "down", tokenId: string, now: number): Promise<void> {
  const leg = side === "up" ? this.state.cycle.upLeg : this.state.cycle.downLeg;
  const remainingCapacity = roundShares(this.config.sharesPerSide - leg.filledShares - leg.orderSize);
  if (remainingCapacity <= 0) return;
  if (leg.orderId) return;
  const price = await this.computeEntryPriceForLeg(side, tokenId);
  const stake = roundUsd(remainingCapacity * price);
  await this.runtime.budget.reserve(stake, `btc15m-hedge-${side}-buy`);
  const response = await this.runtime.placeLimitOrder({ tokenId, side: "buy", price, size: remainingCapacity });
  leg.orderId = extractOrderId(response);
  leg.orderPrice = price;
  leg.orderSize = remainingCapacity;
  leg.orderStatus = "open";
}
```

```ts
private async settleExpiredMarket(now: number): Promise<void> {
  const market = this.state.market;
  if (!market) return;
  const startPrice = this.state.marketStartBtcPrice ?? await this.fetchMarketStartPrice(market);
  const settledBtcPrice = await this.runtime.fetchBtcPrice(market.endTimeMs);
  if (startPrice === null || settledBtcPrice === null) return;

  const winningSide = settledBtcPrice > startPrice ? "up" : "down";
  const pairedPayoutUsd = roundUsd(this.state.cycle.pairedShares * 1);
  const pairedCostUsd = roundUsd(
    this.state.cycle.pairedShares * ((this.state.cycle.upLeg.avgEntryPrice ?? 0) + (this.state.cycle.downLeg.avgEntryPrice ?? 0))
  );
  const pairedPnlUsd = roundUsd(pairedPayoutUsd - pairedCostUsd);
  await this.runtime.budget.addFunds(pairedPayoutUsd, "btc15m-hedge-settlement");
  await this.recordCompletedCycle("paired_hold", pairedPnlUsd, winningSide, now);
}
```
